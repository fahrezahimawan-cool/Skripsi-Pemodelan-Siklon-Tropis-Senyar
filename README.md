# Skripsi-Pemodelan-Siklon-Tropis-Senyar
Seluruh skrip pemrograman Python berformat Jupyter Notebook (.ipynb) yang digunakan untuk mengekstraksi data satelit, melakukan perhitungan statistik (Cross-Correlation dan Diagram Taylor), serta memvisualisasikan data spasio-temporal dalam penelitian ini dilampirkan secara digital pada repositori GitHub.
